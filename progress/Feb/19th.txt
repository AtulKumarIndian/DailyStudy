When encapsulating classes, a good convention is to declare all variables of a class private. 
This will restrict direct access by the code outside that class.

One has to implement public methods to let the outside world communicate with this class. 
These methods are called getters and setters. We can also implement other custom methods.

Encapsulation means to hide data using getter and setter functions. 

Abstraction means to hide implementation using interface and abstract classes.
